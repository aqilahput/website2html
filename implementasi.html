HTML

Belajar HTML Bersama

Apa itu HTML?
HTML (Hypertext Markup Language) adalah kode-kode tag yang menginstrusikan browser untuk menghasilkan tampilan sesuai yang diinginkan.

Sejarah HTML
1. HTML 2.0 disetujui sebagai standard pada 22 September 1995.
2. HTML 3.2 pada 14 Januari 1996.
3. HTML 4.0 18 Desember 1997.
4. ISO HTML (ISO/IEC 15445:2000 "ISO HTML") berdasar pada HTML 4.01 Strict-15 Mei 2000.
5. XHTML 1.0 (Gabungan HTML & XML) pada 24 Januari 2000.
6. XHTML 1.1 pada 16 Februari 2007.

Ciri-ciri Dokumen HTML
1. Eksistensi file berupa .htm atau .html.
2. Non case sensitive.
3. Terdiri atas tag-tag pembuka dan tag-tag penutup (sebagian tag tidak memiliki penutup).
4. Tag-tag saling berpasangan dan bersarang.

Struktur HTML
1. Document Information
- Style = mengatur komponen yang ditampilkan ke dalam browser.
- Page tittle = Judul dari halaman web.
- Scripting = Tempat client-side script.
- Meta tags, description, & keywords = Mempermudah search engine dalam melakukan indexing.
2. Document Header
3. Document Body
- Bagian ini memuat teks, gambar, link, server slide script, dan multimedia and special programmed events.

Penulisan tag
1. Tag pembuka "< >"
2. Tag Penutup ""
3. Single Tag "< />"
